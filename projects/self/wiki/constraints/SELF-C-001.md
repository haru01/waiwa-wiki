# Claude Code でのツール/プロトタイプ開発力

## 制約の記述

- Claude Code を使って、動くツール／プロトタイプを素早く形にできる。評論でなく実装に落とせる。
- 「作ったものが実際に効いた瞬間」を過去に経験している（＝この開発力は既に外界で効いた実績がある）。

## これがどう目的の出発点になるか

- この開発力があるから「実際に効くツールを作ることで支援する」という目的（[[SELF-P-001]]）が立ち上がる。芯（作ったものが効くこと）の直接の土台。
