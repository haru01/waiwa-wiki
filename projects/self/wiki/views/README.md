# views/ — 生成物（手編集禁止）

このディレクトリのファイルは `/view` スキル（`tools/gen_views.py`）が生成する**再生成可能な生成物**である。

- **手で編集しない。** 記録の修正は目的・制約・試行・内省・意思決定の各レコード側で行い、ビューは再生成する。
- `/view list` — 目的仮説テーブル＋**目的の系譜**図（`leads-to`/`derived-from`/`revises`）＋次に確かめるべき目的。
- `/view board` — 試行（ACT）を1実験として時系列に並べる試行ボード＋現在地。
- `/view relations` — 全関係型のグラフ＋バックリンク索引＋目的↔制約の接地（`grounded-in`）フィット。

Obsidianのグラフビューを使うときは、このフォルダをフィルタで除外すると目的ネットワークが見やすい。
