# sources/ — Raw Sources（不変層・この案件専用）

この案件の生データ（壁打ちセッションのログ・試行の記録・面談メモ・第三者の反応など）を**人間または `/reflect` が置く**ディレクトリ。

## ルール

- **AIは読むだけ。既存ファイルを編集・削除しない。** 一次情報の置き場であり、後から書き換えると歴史性が壊れる（新規追加は可）。
- ファイル名は自由（推奨: `YYYY-MM-DD-<種別>-<説明>.md`）。
- **冒頭に種別タグを必ず書く**（証拠の非対称性の判定に使う）:

  ```
  > **種別**: trial | dialogue | third-party
  ```

  | 種別 | 中身 | 確信度を上げてよいか |
  |---|---|---|
  | `trial` | 現実の試行の記録（やってみた・打診した・断られた） | ○（〈試行〉〈継続〉） |
  | `third-party` | 第三者の生の反応（面談メモ・フィードバック） | ○（〈他者反応〉） |
  | `dialogue` | 壁打ち・質問セッション（エージェントとの対話） | **× 単体では上げない**（〈壁打ち〉。目的文を磨く材料） |

- **架空/脳内シミュレーションデータを置く場合は、冒頭に「架空・実証拠として扱わない」と明記する**（確信度は上限8）。
