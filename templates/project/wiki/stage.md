# 探索モード

```yaml
current-mode: 探索
updated: YYYY-MM-DD
```

現在の探索モードは **探索**（explore）。目的形成は「確信を高める（深掘り／exploit）」と「探索域を開き直す（探索／explore）」の二相を往復する（v2 §6）。

- **探索**: 目的仮説の種を広げる。`/surface` で新しい P を生む。確信度は低く保つ。
- **深掘り**: 手応えのある P を外界にぶつけて確信を育てる。`/reflect` で試行を紐づけて確信度を上げる。

フェーズ1では停滞検知は**手動**（週次で `explore-log.md` と `/view` を見て、深掘りに偏って停滞していないか本人が判断する）。フェーズ2で `/stall-check` が自動化する。
