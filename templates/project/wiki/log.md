# 活動ログ（追記専用・grep可能）

形式: `## [YYYY-MM-DD] <type> | <ID> <要約> → <影響目的と確信度変化>`

type は `purpose`（目的仮説の起票/更新）／ `打診` `やってみる` `面談` `観察` `self-reflection`（試行 ACT）／ `reflection`（内省 REF）／ `constraint`（制約 C）／ `decision`（意思決定 DEC）／ `pilot`（確信度の引き下げ）・`lemonade`（探索再開）／ `lint` のいずれか。

過去行は編集しない（追記のみ）。例: `grep "decision" log.md` で意思決定だけを抽出できる。確信度変化は `P-001 1→5` のように書く。
